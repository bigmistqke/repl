# @bigmistqke/repl

[![pnpm](https://img.shields.io/badge/maintained%20with-pnpm-cc00ff.svg?style=for-the-badge&logo=pnpm)](https://pnpm.io/)

repl with typescript support

> **Note** After using this template, you have to search and replace all `@bigmistqke/repl` and similar strings
> with appropriate texts.
>
> `@bigmistqke/repl` should be a **kebab-case** string representing the name of you monorepo.
>
> `repl with typescript support` should be a **Normal case** string with the description of the repository.
>
> `bigmistqke` should be a **kebab-case** string from your profile URL.

## Quick start

Install it:

```bash
npm i @bigmistqke/repl
# or
yarn add @bigmistqke/repl
# or
pnpm add @bigmistqke/repl
```

Use it:

```tsx
import @bigmistqke/repl from '@bigmistqke/repl'
```
